# ChardhamYatra

Visual **Chardham + Panch Prayag Yatra** planner for May–June 2026.

## Features

- 12-day itinerary aligned with Travel Key Solution agent plan
- Hour-by-hour schedule with flight ticket details (AMD → DEL)
- Health care notes for sugar & BP patients
- Medicine, packing, trek & Panch Prayag guides
- Animated UI with temple photo backgrounds

## Open locally

```bash
# Open in browser
xdg-open index.html
```

Or double-click `index.html` in the `ChardhamYatra` folder.

Temple photos are bundled in `assets/images/` — no internet needed for backgrounds.

## GitHub Pages

After pushing, enable **Pages** in repo settings → deploy from `main` branch → root.

Live URL: `https://saurabhrathod35.github.io/ChardhamYatra/`
